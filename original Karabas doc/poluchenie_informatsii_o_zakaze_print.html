Получение информации о заказе

 
 
 
  Команда возвращает информацию о текущем или конкретном заказе.
 
GetOrderCommand.cmd
Параметры
__auth
Ключ авторизации
Обязательный параметр
order
ID заказа
Если нужно получить информацию по конкретному заказу
format
По умолчанию JSON, или XML если передать format=xml
Формат выдачи данных
Ответ
orderId
ИД заказа
 
amount
Сумма
Равно 0, пока не произошло подтверждение заказа.
status
Статус
0 - резерв
1 - оплачен
2 - отменен
3 - ожидает оплаты
4 - возврат
expires
Дата истечения заказа
Возвращается только в случае, если заказ находится в состоянии резерва
currency
ИД валюты
 
currencyName
код валюты
 
orderItemsCount
кол-во билетов
 
orderItemsTotal
стоимость билетов
 
cashServiceTotal
стоимость билетов
 
clientId
ИД клиента
 
ClientName
ФИО клиента
 
deliveryType
тип доставки
 
paymentType
тип оплаты
 
deliveryAddress
адрес доставки
 
items
Массив билетов|Ниже следуют аттрибуты объектов массива
id
Идентификатор билета
 
pid
Идентификатор места
 
barCode
Штрих-код
 
event
ИД мероприятия
 
eventName
Мероприятие
 
eventDate
Дата мероприятия
 
zone
Зона
 
row
Ряд
 
number
Место
 
price
Цена
 
totalAmount
Стоимость билета
=цена билета + сервисный сбор - скидка
(price+cashServiceAmount-bonusDiscountAmount)
bonusAmount
Бонус
 
bonusValuedDescription
Описание бонуса
 
bonusDiscountAmount
Скидка
 
bonusDescription
Описание скидки
 
cashServiceAmount
Сервисный сбор
 
cashServiceDescription
Описание сервисного сбора
 
HasProducts
Признак товара
1 - Товар
0 - Билет
currency
ИД валюты
 
currencyName
Валюта
 
salesLine
Линия продаж
 
eventPacket
ИД пакета
 
EticketUrl
ссылка на ПДФ файл электронного билета (если разрешено)
 
deliveryType
Способ доставки
ID
deliveryAddress
Способ доставки
string
paymentType
Способ оплаты
ID
 
IsETicketPassportRequired
Паспортные данные на электронном билете обязательны
1 / 0
 
IsThermoTicketPassportRequired
Паспортные данные на термобилете обязательны
1 / 0
 
ClientInfoRequired
Данные клиента обязательны на электронном билете
1 / 0
 
 
Пример кода:
function getOrder() {
    var src = " http://kassir.kartina.tv/GetOrderCommand.cmd"
    $.post(src, { __auth: _auth }, function (data) {
        refreshCart(data);  //обновим корзину
            _orderId = data.orderId; //запомним ИД заказа
    }, "jsonp");
}
 
function refreshCart(order) {
    $("#ticketsContainer").html("");
    var table = $("<table></table>").appendTo("#ticketsContainer");
 
    for(var i=0;i<order.items.length;i++) {
        var item = order.items[i];
        var tr = $("<tr></tr>").appendTo(table);
 
        $("<td></td>").html(item.zone).appendTo(tr);
        $("<td></td>").html(item.row).appendTo(tr);
        $("<td></td>").html(item.number).appendTo(tr);
        $("<td></td>").html(item.price).appendTo(tr);
    }
}
