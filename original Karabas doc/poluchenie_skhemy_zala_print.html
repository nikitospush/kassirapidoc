Получение схемы зала

 
 
Для начала необходимо получить полную схему зала с помощью вызова команды
GetFlashHallDataCommand.
 
GetFlashHallDataCommand.cmd
Параметры
__auth
Ключ авторизации
Обязательный параметр
event
ИД мероприятия
Обязательный параметр
Ответ
Тег hall
Id – идентификатор зала
Name – наименование зала
Swf- не используется
Scale – масштаб
Тег zone
Описание зон (секторов)
Id- идентификатор зоны
Name- название зоны
Color – цвет зоны
Тег p
Описание места
Id- идентификатор места
Row –ряд
Num - номер
Zone – зона (см. Тег zone)
Bi – базовый индекс
LinkToZone – для больших залов, ссылка на дочернюю зону со своими местами
X – х-координата
Y – y-координата
Price – стоимость
Для залов Canvas
Тег background
Описание подложки
src - ссылка на подложку (в случае отсутствия подложки - пусто)
x - координата по оси Х верхнего левого угла
у - координата по оси У верхнего левого угла
width - ширина подложки
height - высота подложки
Тег labels
Описание меток
х - координата по оси Х
у - координата по оси У
fontSize - размер шрифта
fontFamily - имя шрифта
isBold - Жирный (1/0)
isItalic - Курсив (1/0)
