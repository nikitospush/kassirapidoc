Получение профиля клиента

 
 
 
  Данная команда получает данные о клиенте. Клиент должет быть авторизован через ClientLoginCommand.cmd
 
ClientInfo.cmd
[GET]
Параметры
__auth
Ключ авторизации
Обязательный параметр
clientProfile
возвращает профиль клиента [0, 1]
Необязательный параметр
returnBonuses
[0, 1]
возвращать состояние бонусного счета (если clientProfile=1)
Необязательный параметр
returnTickets
[0, 1]
возвращать все резервы и оплаченные билеты клиента (если clientProfile=1)
Необязательный параметр
returnSaleTickets
[0, 1]
возвращать билеты в перепродаже (если clientProfile=1)
Необязательный параметр
format
По умолчанию JSON, или XML если передать format=xml
Формат выдачи данных
Ответ
errormessage
