Очистка текущего заказа

 
 
 
  Данная команда очищает текущий заказ и создает новый, возвращая его идентификатор.
 
ClearCurrentOrderCommand.cmd
Параметры
__auth
Ключ авторизации
Обязательный параметр
format
По умолчанию JSON, или XML если передать format=xml
Формат выдачи данных
Ответ
order
ИД нового заказа
