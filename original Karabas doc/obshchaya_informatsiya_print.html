Общая информация

 

Централизованная продажа и сопровождение заказов

 
  Система «Kassir: Ticket» построена для централизованной обработки запросов любых других внешних систем и предоставляет актуальные данные всем, кто подключен к ядру с помощью универсального API. Актуальность данных обеспечивается внутренними механизмами системы, что позволяет партнерам вести продажи в режиме реального времени.
 

Механизмы подключения

 
  Для подключения к системе необходимо получить ключ партнера, по которому будет происходить его авторизация. Существует два типа подключения разной степени сложности в реализации:
 
  • Универсальная касса. В вашей системе или на сайте необходимо сделать лишь кнопку или ссылку для перехода на страницу универсальной кассы, где пользователь может оформить заказ. Заказ учитывается в системе учета как партнерский.
  • API. Вся реализация пользовательского интерфейса производится на стороне компании-партнера с использованием API функций «Kassir:Ticket».
 

Авторизация

 
  Для работы с API внешняя система должна пройти авторизацию с использованием ранее полученного ключа партнера. Проходя авторизацию по ключу партнера система выдает временный идентификатор сессии, который в дальнейшем используется при вызовах API -команд.
 

Заказ

 
  Для каждой клиентской сессии на сервере Kassir формируется и хранится заказ. Заказ имеет несколько возможных состояний и определенное время жизни. Если заказ не был подтвержден – он будет автоматически удален в течении установленного лимита времени. Это следует учитывать при реализации собственного механизма обработки заказов с помощью API. Время жизни заказа возвращается в ответе команды получения информации о заказе, а также в ответе команды подтверждения заказа.
 

Команда

 
  Взаимодействие клиентского приложения с бизнес-логикой системы производится с помощью команд. Команда может принимать входящие параметры (например для фильтрации данных) и возвращать результат обратно на клиент в формате JSON или XML. Обычно взаимодействие с командами реализуется с помощью AJAX.
