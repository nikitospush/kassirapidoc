Удаление билета из корзины

 
 
  Удаление места из корзины производится с помощью команды DeletePlaceFromCartCommand.
 
DeletePlaceFromCartCommand.cmd
Параметры
__auth
Ключ авторизации
Обязательный параметр
orderItem
ИД билета
Обязательный параметр
format
По умолчанию JSON, или XML если передать format=xml
Формат выдачи данных
Ответ
Пустой ответ или сообщение об ошибке
