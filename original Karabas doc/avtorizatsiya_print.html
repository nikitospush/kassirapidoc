Авторизация

 
  Авторизация выполняется один раз при первом посещении пользователем сайта компании-партнера.
Сайт должен хранить полученный код авторизации и передавать его во все последующие вызовы команд.
 
LoginCommand.cmd
Параметры
__uid
Ключ API-пользователя
Обязательный парметр
Ответ
__auth
Ключ авторизации
Хранится на сайте компании-партнера
type
Тип партнера
0 - организатор, 1 - реализатор
 
Пример:
 
var src = "http://kassir.kartina.tv/LoginCommand.cmd"
$.post(src,
{
  __pid: "3imNRizOQJF7TStWm3HsEg==",
  __uid: "3hhsdgfyyrikmfhyfbf663hd"
}, function (data) {
    _auth = data.__auth;
    loadHall();
    getOrder();
}, "jsonp");
