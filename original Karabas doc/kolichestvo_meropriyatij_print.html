Количество мероприятий

 
Команда возвращает количество мероприятий, доступных партнеру, с группировкой по месяцам .
 
GetPartnerEventsStatistic.cmd
 
 
Параметры
__auth
Ключ авторизации
Обязательный параметр
from, to
Фильтр по дате начала мероприятия
Обязательный параметр, yyyy/mm/dd
City
Фильтр по городу
(можно указать несколько через запятую)
ID
Ответ
monthlyEvents
Массив мероприятий Ниже следуют аттрибуты объектов этого массива
 
eventsCount
Количество мероприятий
 
month
Первый день месяца проведения мероприятия (формат Дата)
"2016/04/01"
