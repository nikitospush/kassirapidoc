Использование виджета

 
 
 
  Виджет предназначен для резервирования и продажи билетов, продажи электронных билетов с оплатой через системы онлайн-платежей.
Принимаемые параметры виджета:
 
Tickets.aspx
Параметры
__uid
Ключ пользователя (сайта)
Обязательный параметр
__auth
Ключ сессии
Если заранее был получен при работе с АПИ
event
ID мероприятия, которое необходимо отобразить
Обязательный параметр
lang
Язык интерфейса по умолчанию
Варианты: ru, ua, en
Вызвать кассу можно двумя способами:
  • прямым переходом на страницу виджета
  • открытие виджета в iframe
Пример ссылки на страницу виджета: http://kassir.kartina.tv/PublicAPI/WidgetStep1.aspx?event=<ID мероприятия>&__uid=<полученный UID при заведении сайта>
