Введение

      Данный документ описывает идеологию и порядок использования API «Kassir:Ticket» для продажи билетов на зрелищные мероприятия в партнерских системах и предназначается для ИТ-специалистов, занимающихся интеграцией своих сайтов и информационных систем с централизованной системой продаж билетов.
 
API «Kassir:Ticket» позволяет:
  • Элемент ненумерованного списка
  • Получать список доступных к продаже мероприятий;
  • Использовать готовые схемы залов для онлайн-продажи билетов;
  • Подключать собственные схемы залов, пользуясь общей системой резервирования;
  • Резервировать, оплачивать, отменять билеты;
  • Получать информацию по заказам;
  • Работать с корзиной билетов;
  • Использовать универсальную кассу для продажи билетов.
