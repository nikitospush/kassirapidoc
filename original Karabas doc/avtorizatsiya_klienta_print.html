Авторизация клиента

 
 
 
  Данная команда авторизует клиента.
 
ClientLoginCommand.cmd
Авторизация клиента
Параметры
__auth
Ключ авторизации
Обязательный параметр
authProvider
Тип авторизации [ sms, email, fb, vk ]
Обязательный параметр
clientProfile
возвращает профиль клиента [0, 1]
Необязательный параметр
returnBonuses
[0, 1]
возвращать состояние бонусного счета (если clientProfile=1)
Необязательный параметр
returnTickets
[0, 1]
возвращать все резервы и оплаченные билеты клиента (если clientProfile=1)
Необязательный параметр
returnSaleTickets
[0, 1]
возвращать билеты в перепродаже (если clientProfile=1)
Необязательный параметр
logout
Выйти из текущей сессии
Необязательный параметр
mobile=1
Увеличенное время жизни мобильной сессии. Используется для мобильных приложений.
Необязательный параметр

password
