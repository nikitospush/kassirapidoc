Валюты

 
 
 
  Данная команда возвращает все валюты, которые в данный момент есть в системе Kassir.
 
 
GetCurrenciesCommand.cmd
GET
Параметры
__auth
Ключ авторизации
Обязательный параметр
Ответ
errormessage
Если произошла ошибка во время выполнения команды
 
totalCount
Общее кол-во записей в ответе
int
rows
Массив записей
id
ID типа
id
IsoCode
Код ISO валюты
Например: "USD", "UAH", "EUR"
Iso4217Code
Числовой код ISO 4217
Например: "840", "978", "980"
FullName
Полное название валюты
 
Например: "Гривня", "Евро"
ShortName
Сокращенное название валюты
Например: "€", "$", "грн"
