Загрузка готового компонента

 
 
 
Для загрузки готовой схемы зала вам потребуется выполнить следующий код:
 
function loadHall() {
var src = "http://kassir.kartina.tv/HallViewCanvas.aspx
?width=800&height=600
&includeJs=1
&__auth=" + _auth + "                       
&event=" + _event;
        $("#hallContainer").attr("src", src);
}
          <div>
  <iframe id="hallContainer" style="width:850px;height:650px;border:0;"></iframe>
</div>
 
HallViewCanvas.aspx принимает следующие параметры:
    
  •   __auth – ключ авторизации, полученный с помощью команды авторизации
  • width, height – ширина и высота схемы зала
  • event- идентификатор мероприятия
  • includeJs - подключает JQuery , если указан
  • lang - выбор языка (ua, ru, en, de)
  • showLegend - показывать легенду (0,1)
