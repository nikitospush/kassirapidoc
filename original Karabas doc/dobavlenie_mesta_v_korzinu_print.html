Добавление места в корзину

 
 
  Добавление места в корзину происходит при нажатии мышью по месту на схеме зала и последующим вызовом команды ChangePlaceCommand. Для обновления корзины используйте GetOrderCommand.
 
ChangePlaceCommand.cmd
Параметры
__auth
Ключ авторизации
Обязательный параметр
event
ИД мероприятия
Обязательный параметр
pid
Id места
Обязательный параметр
returnStatus
Возвращать ответ с добавленными билетами (returnStatus=1) или ответ пустой(returnStatus=0)
Необязательный параметр
format
По умолчанию JSON, или XML если передать format=xml
Формат выдачи данных
Ответ
status
текущий статус места
0 - свободно
1 - продано
2 - текущий заказ
3 - не используется
4 - в резерве
nextStatus
Следующий статус места
0 - при следующем вызове место будет свободно
1 - место заблокировано, следующий вызов не позволит добавить в корзину
2 - текущий заказ, место нельзя добавлять в заказ
3 - не используется
4 - в резерве
placeId
Идентификатор места
 
eventId
Идентификатор мероприятия
 
orderId
Ид текущего заказа
 
free
кол-во доступных мест
 
Важно! После добавления мест в корзину и выбора типа оплат, нужно подтвердить заказ (перевести в статус резерва).
В противном случае срок жизни неподтвержденного заказа - 15 минут с момента последнего действия.
