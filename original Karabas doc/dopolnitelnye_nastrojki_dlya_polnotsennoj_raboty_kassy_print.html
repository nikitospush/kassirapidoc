Дополнительные настройки для полноценной работы кассы

 
 
 
  • Способы доставки и оплаты, которые предоставить клиенту
  • Реквизиты для платежных систем
  • Список касс, который будет отображаться при выборе клиентом способа доставки «Забрать на кассе»
  • Банковские реквизиты, которые будут отображаться при выборе клиентом способа оплаты «Банковский перевод»
  • Почтовые настройки. SMTP-Сервер, шаблоны писем для клиентов. Есть возможность разрешить отправку письма только в конкретных случаях
  • Логотип для размещения на виджете и в личном кабинете клиента. Размер не больше 224x66 px
